# Modern Developer Portfolio

A modern, responsive developer portfolio for showcasing projects, skills, and online presence.

---

## Features
- Responsive layout  
- Modular React components  
- Glassmorphism UI design  
- AMOLED black theme with multi-color accents  
- Fast development workflow  
- Easy customization  

---

## Tech Stack
- Vite  
- React  
- TypeScript  
- Tailwind CSS  
- shadcn-ui  

---

## Local Setup

Clone the repository  
`git clone https://github.com/YUKIHANA-REALMS/modern-portfolio`

Move into the project directory  
`cd modern-portfolio`

Install dependencies  
`npm install`

Start development server  
`npm run dev`

---

## Production Build

Build the project  
`npm run build`
